PDFData: { cache: boolean; code: string; name: string; offset: number | string; url: string }

A data objection containing properties of a user-created PDF file. This is the data contained within a PDFoundry_PDF entity.

Type declaration

  • code: string

  • name: string

    The name of the PDF

  • offset: number | string

  • url: string
