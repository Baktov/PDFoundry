Static openURL

  • openURL(url: string, page?: number, cache?: boolean): Promise<Viewer>

  • Open a URL as a PDF.

    Parameters

    • url: string

      The URL to open (must be absolute).

    • Default value page: number = 1

      Which page to open to. Must be >= 1.

    • Default value cache: boolean = true

      If URL based caching should be used.

    Returns Promise<Viewer>

  • getPDFData(comparer: ItemComparer): PDFData | null

    Returns PDFData | null

Static getPDFDataByName

  • getPDFDataByName(name: string, caseInsensitive?: boolean): PDFData | null

    • Default value caseInsensitive: boolean = true

      If a case insensitive search should be done.
